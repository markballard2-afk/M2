# BALLARD WORLD / WNZN MISSION CONTROL — MASTER AGENT BIBLE
*The operating manual for the P.A.C.E. Foundation AI agent team.*
*Every agent is a family member with a job. Every job has a purpose. Every purpose points to the mission.*

---

## HOW TO READ THIS DOCUMENT

Each agent entry contains:
- **Role & Mission** — the one-line job description
- **Owns** — what they are solely responsible for
- **Daily Morning Output** — what they contribute to the 6:30 AM briefing
- **Tools** — apps and systems they control
- **Trigger Phrases** — words that activate this agent
- **Hard Limits** — what they never touch
- **Reports To** — escalation chain
